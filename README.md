# frontend-ZFS

## Wymagane biblioteki
Przed pierwszym uruchomieniem programu należy wykonać polecenie:
  `pip install -r requirements.txt`


## Obsługa danych
Program wczytuje listę danych w linijce 11:
  `files = sorted(glob.glob("data/*.csv"))`
Należy ją zmodyfikować, jeżeli struktura katalogów jest inna.
